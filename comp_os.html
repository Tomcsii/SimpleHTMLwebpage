Importance of a Computers Operating System
This is the subwebpage dedicated to explain the importance of an Operating System in a computer.


Operating systems are the most important part of the computer other than the hardware, without an operating sysrtem a computer would be usless as opertaing systems help carry out some of the most important tasks when it comes to computers, Why an Operating System is important:
  • The operating system is program that lets you interact with a computer

  • Communicatin link between the user and the computer and helps the user run applications and programs

  • An operating system manages your computer needs, It does this by: finding resources, applying hardware management and providing necessary services. Operating systems are essential for computers to be able to function as well as they do .

  • An OS communicates with various parts of your computer, it sends information from computer hardware and the disierd application in order to carry out the tasks you request

  • Operating system and computer hardware form a complete system that can determine what your computer can do

  • The operating system manages of all of the software and hardware on the computer

  • The operating system makes sure each running program gets what it needs


Some of the tasks the Operating system preform:
  • Recognize mouse click requests

  • Register keyboard commands

  • Deliver visual content

  • Organize content

  • Connect to printers, ports, drives, etc.

Some of the hardware the OS communicates with are:
  • Mouse

  • Printer

  • Monitor

  • Disk drive

  • Keyboard
