Common User Interface Elements and Their Functions
This is The subwebpage dedicated to explain some of the common user interface elements of a computer and their functions.


Computer User Interface Elements Function of User Interface Element
Desktop The desktop is the main screen that you are presented with once you log into your computer, the desktop is bascially a "home page" as everything can be accesed and veiwed on the desktop including; shortcuts, the toolbar, folders, rycyle bin, etc
Toolbar The Toolbar is stretching across the entire horizontal axis at the bottom of your screen, It houses a strip of diffrent icons symbols and shorcuts with unique functions.
Start Menu The Start menu is located in the bottom left of your screen as a small windows logo, when pressed it will reveal the start menu, you can also click the windows key on your keyboard to reveal the start menu, The start menu is a personalized menu where you can acces any of your installed programs, the settings, user profile, file manager,pictures and almost anything else. some say it is the primary menu to locate your installed programs.
File Manager The file manager located on your toolbar as a stock icon is where you can locate any and all of your files, downloads, pictures etc and anything else stored on your harddrive
Icons Icons on your computer can be found on the toolbar the start menu the desktop and almost anywhere else on your computer, Icons are used to tell the user what the application or what file/document type it is whitout having to read anything and or opening anything.
Shortcuts A shortcut is a handle that allows a user to find a file or applictaion in a diffrent directory or folder quickly and with ease to whereever the shortcut is located.
Folders Folders are an organization and storage tool where many files can be placed to help group or organize them, a folder can also contain other folder and so on directory and pathways are used to find a certain file or application that is within a certain file location.
Search Bar a Search bar located in the bottom left near the start menu icon is used to search up any files or applicaitons or anything else located on your computer quickly and with ease if osmething is not found on your computer it will automatically bring search results form the internet for you.
Rycyle Bin The Rycyle bin on your computer is usually found on your desktop and can be found using any other file browsing method (file manager, start menu, etc) It is used to get rid of any unwanted files you would like to "trash" away, emptying the rycyle bin will permanently delete the file
Assistant The assistant or Cortana is found right of the search bar and is like any other ai assistnat (siri, google) assistant can help you set reminders, timers, remind you of notifications, she can also adjust a handful of settings
Hiddin Apllication, Sound, and Language Hiddin Applications Sound and Language are all located in the bottom right of your taskbar, The Hidden applications are where applications that are running in the background currently are kept hidden away examples may be spotify steam your internet acces etc, the sound option is about changing the volume of sound the computer is outputting from the speakers and the language is what laguage preset your keyboard will be read in.
Date Time, Notification Date/Time and the notification tab are all found in the bottom right of your task bar, the date and time will display the date and time currently, you can also change the date and time from this shortcut, the notification tab where any incoming notifications from programs on your computer will be kept until you have either removed or read them ex: email or messangers
