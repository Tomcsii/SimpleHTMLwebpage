Digital Camera a cmaera that records and stores digital images that can be displayed on a computer. Input
Video Camera A Camera for recording images on videotape or for transmitting them onto a monitor it can aslo be used to transmitting them onto a monitor screen. Input
Microphone A microphone is an input device used for recording sounds that can then be processed by a computer and can be inputed. Input
Printer A printer is a output device that can be used for printing things like text or images onto paper from a computer. Output
External Hard Drive An external hard drive is the same as a normal hard drive (Being a storage device with a hard disk on it) the main diffrence is that it is portable and can easily be atttached and de attached from a computer. Both
